Lua for ITGmania

Hello there! In these pages, you'll find my Lua for ITGmania guide, comprised of various chapters organized under the headers on the left at the bottom of this page.

If you're unsure of where to start, check out the chapter on What Are Actors? to get a sense of how Lua files are organized in ITGmania.

ITGmania's complete Lua API documentation can be found in its Lua.xml file, which ships with releases of ITGmania in the ./Docs/Luadoc/ directory and can be viewed in most web browsers. I have also hosted a copy as part of this website: Lua For ITGmania: API Reference.

You can find additional resources for learning about Lua and SM5 scripting on the Other Resources page.
